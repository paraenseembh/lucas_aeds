Questão 4 - Inversão de String (ITERATIVO)
============================================

Crie um método ITERATIVO em C que recebe uma string como parâmetro e retorna a string invertida.

Na saída padrão, para cada linha de entrada, escreva uma linha de saída com a string invertida.

EXEMPLO:
  Entrada: abcde
  Saída:   edcba
