Questão 2 - Alteração Aleatória (ITERATIVO)
============================================

Crie um método ITERATIVO que recebe uma string, sorteia duas letras minúsculas aleatórias
(código ASCII >= 'a' e <= 'z'), substitui todas as ocorrências da primeira letra na string
pela segunda e retorna a string com as alterações efetuadas.

Na saída padrão, para cada linha de entrada, execute o método desenvolvido nesta questão e
mostre a string retornada como uma linha de saída.

EXEMPLO DE ENTRADA:           EXEMPLO DE SAÍDA:
o rato roeu a roupa do rei    o rqto roeu q roupq do rei de romq
de roma
e qwe qwe qwe ewq ewq ewq     k qwk qwk qwk kwq kwq kwq
FIM

Observações:
- A classe Random do Java gera números (ou letras) aleatórios.
- seed é a semente para geração de números aleatórios.
- Nesta questão, por causa da correção automática, a seed será QUATRO (4).
- A disciplina de Estatística e Probabilidade faz uma discussão sobre "aleatório".

Exemplo de código para gerar uma letra minúscula aleatória:

    Random gerador = new Random();
    gerador.setSeed(4);
    System.out.println((char)('a' + (Math.abs(gerador.nextInt()) % 26)));
